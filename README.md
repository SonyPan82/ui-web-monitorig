UI-WEB-MONITORING

Voici mon projet étudiant réalisé à l'École Centrale de Nantes.
Ce projet avait pour objectif de créer un outil de monitoring en temps réel de sites web. Il constituait notre examen final.

Cette version est une reconstruction retravaillée du projet original, réalisée à partir des fichiers que j'ai retrouvés sur mon ordinateur personnel.

J’y ai également réintégré l’interface que j’avais imaginée au début du projet, mais qui n’avait finalement pas été retenue dans la version finale réalisée en groupe.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
